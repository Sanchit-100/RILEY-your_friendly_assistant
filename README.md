# RILEY-your_friendly_assistant
![Image](Resources/R.I.L.E.Y..png)

## Overview
RILEY acts as an assistant with the following abilities:-
1. Set Reminders
2. Take a picture
3. Play a Quiz
4. Search Information
5. Translate Multiple Languages
